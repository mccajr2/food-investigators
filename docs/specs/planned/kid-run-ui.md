## Problem

The iPad tasting run is already simple enough for a 5–6 year old, but the
visual surface still reads like a generic CSS/component kit. Parents want that
same calm, easy flow with a look that feels like Food Investigators — playful
and kid-facing without adding clutter or reading load.

## Non-goals (sketch)

- Changing the outcome question set or complete API
- Native iOS theming (`run-tasting-session-ios`)
- Dense “dashboard” chrome or more steps in the ritual
- Overhauling Plan / History / Foods laptop chrome in the same PR (run-first)
